
###代码改动 1.新增了package  cal:globals说明

```

以前的router在   ／lib下面
新的router放到了client／lib／router下面， 并且每个人根据自己的模块创建自己的独立的router文件

由于在meteor中 client／lib／中的内容优先于／lib里的内容，这样router的加载在基本处于目前最靠前的位置了。


要确保APP Cal等业务逻辑相关的全局变量在之前定义，需要建立更深的文件夹，或取排序靠前的命名，并且以后新建文件或文件夹都要非常小心

总之有优先加载需求的文件其同级文件以及父级文件夹的同级文件夹命名都要十分谨慎，
综合考虑作为package最省事

```
